Tarotify Wrapped

Web Design and Development
Project Overview
A Django-based web app showcasing personalized Spotify listening insights. Features include dynamic Spotify Wrapped slides, user accounts for saving past summaries, mobile-friendly design, and interactive elements like games and Duo-Wrapped comparisons. Developed using Scrum methodology, CI/CD pipelines, and hosted for public access.
